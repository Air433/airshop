  #### 模块说明
  ```
  airshop
  ├── airshop-search ── elasticsearch全文检索服务
  ├── airshop-common ── 公共模块
  ├── air-api-gateway --网关服务
  ├── air-shopgoods-web ── 静态页面服务
  ├── airshop-cart ── 购物车服务
  ├── airshop-item ── 商品服务
  ├── airshop-order ── 单据服务
  ├── airshop-auth ── 权限校验服务
  ├── air-registry ── 注册中心
  ├── air-upload ── 文件上传服务
  ├── airshop-user ── 用户服务
 ```
